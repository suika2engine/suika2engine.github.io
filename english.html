Four Glances
at Suika2

Open-Source

As an open-source project, Suika2 allows not only the development team, but the community to contribute to its continued enhancement and modernisation.

Cross-Platform

With support for cross-platform reading and development, Suika2 allows developers and players the ability to enjoy stories across a range of devices.

Simple to Use

With a 'plain-English' markup language and an intuitive, commented configuration setup, Suika2 allows developers to focus on their stories, not syntax.

Global

Suika2 is capable of effortlessly displaying text in Japanese, English (Latin Scripts), Chinese, Russian and Ukrainian, Korean, and many other languages.

Update 2.11.7

Improved Multilinguilisation Support

A new system utilising language specifiers has been introduced to further provide native multi-language support. 

Windows x Unicode

Suika2 for Windows now utilises unicode information to provide a better language experience.

Minimum macOS Version

The minimum macOS version supported by Suika2 has been raised to macOS 10.13 High Sierra.

Minor Bug Fixes

A number of bugs have been fixed in this update, providing a smoother, cleaner user experience.
