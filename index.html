Why Choose Suika2?

Cross-Platform

Suika2 runs as a native application on Windows, Mac, IOS, Android, and other platform so you can create and play without the headache.

Diverse

The Suika2 Development Team and community have memebers from all across the world, with information available in Japanese, English, and Chinese.

Open-Source

Suika2 is an open-source project hosted on GitHub. Found a bug, have an issue, or want to request a feature? Open an issue or join the Discord and let us know!

Easy to Use

Suika2 scenario files (scripts) use Suika2Script, an efficent visual-novel markup that allows game developers and authors to focus on writing, rather than syntax.
