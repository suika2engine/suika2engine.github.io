Suika2.11.9 | Latest Update: 17th December, 2022

1. Installation and Building from Source

Learn how to install or build Suika2 from its source so that you can begin creating your visual novel or adventure game!

1.5. Supported Formats

Learn what formats Suika2 supports as well as what their extensions and usages are. Formats may be updated between Suika2 versions, so make sure to check here for the latest supported formats.

2. Setup and Configuration

Learn what settings and configurations are included out-of-the-box as well as how you can make changes to them and their behaviour.

3. Suika2Script and Commands

Learn about Suika2's markup language, Suika2Script that serves as the building blocks of every game created with the engine. This is a good place to check your syntax and new commands.

4. Creating Graphical User Interfaces

Learn how to correctly define (create and edit) and call GUI files to create menu-screens, UI, and a host of other user-interactable elements.

5. Suika2 Pro for Creators

Learn how to use Suika2 Pro for Creators, Suika2's sister application, to prepare your game for distribution and text the flow or logic of your scenario and script files.
