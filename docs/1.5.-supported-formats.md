# 1.5. Supported Formats

This article aims to provide clarification regarding the formats supported by Suika2.

## Image and Video Formats
Suika2 supports a number of visual formats, these include:

* PNG `(.png)`
* JPG `(.jpeg or .jpg)`

* Windows supports these video formats:
  * `.wmv`
  * `.avi` (supports H.264 + AAC)
* MacOS supports these video formats:
  * `.mp4` (supports H.264 + AAC)
  * `.move` or `.qt` (Quick Time File Format, supports H.264)
* Linux supports these video formats:
  * `.mp4` (supports H.264 + AAC)

**Note:** There are plans to implement `.mp4` for Windows in the future (Most likely when Windows 8.1 reaches EOL).

## Audio Format
Suika2 currently supports one audio format, that being the Ogg Vorbis `(.ogg)` (44.1kHz, stereo or monaural) format.

## Font Formats
Suika2 supports a number of font formats, these include:

* TrueType `(.ttf or .ttc)` – Suika2's default font format.
* OpenType `(.otf or .ttc)` – Suika2 readable font format.

**Note:** Either font format is accepted by Suika2, and a combination can be used.
